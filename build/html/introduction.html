    Introduction¶

    This document describes the actions required by a director to initiate, run and score a duplicate event on BBO.

    Resources¶

    Before you start you will need:

    1. the club’s BBO director’s login (vEBU211328) and the related password;

    2. the club’s BridgeWebs password;

    3. your personal EBU number and password.

    (The latter two are needed for scoring only.)

    Documentation¶

    This document is divided into sections that describe (in detail) the functions involved in running a BBO tournament:

    1. Manage the club’s on-line players;

    2. create a tournament;

    3. run the tournament;

    4. keeping the bbo names file up to date;

    5. scoring.

    These functions are quite independent and do not need to be undertaken by the same person.

    The Browser section describes setting up your browser for scoring and need only be undertaken once.
