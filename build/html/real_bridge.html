    Real Bridge¶

    Creating a Tournament¶

    1. Navigate to the Real Bridge website

    2. Click on Club Admin (at the top right of the page)

    3. Enter the Club’s Username and Access Key and click on Login

    4. In the Club Admin page, set the session date and give the session a decription (no spaces - letters and numbers only)

    5. click on Start session and a line will be created in the session list

    6. Email the Player link to the Website administrator

    7. Email the Director link to the director and the Deputy director link to any deputy directors

    Setting up a Tournament¶

    1. Click on the Director link

    2. Click on Login

    3. Give the session a title

    4. Click on Session settings

    5. Make the relevant changes. The number of boardas per round and the movement might need adjusting when the tournament is due to start.

    6. Add some tables and wait until you are ready to Open the session.

    7. In advanced settings, update the EBU info, MPs, Club name and Number

    Opening the session¶

    Opening the session allows player to sit at the tables. This can be done at any time, although the earlier you start the longer people have to move around and chat before the game.

    Starting the session¶

    Immediately before the start of the session it might be necessary to adjust the movement and the number of boards per round

    Tables Boards / Round Rounds Boards Played Movement
    2 6 3 18 Howell
    3 4 5 20 Howell
    4 3 7 21 Howell
    5-8 2 9 18 Howell
    9+ 2 9 18 Mitchell

    (for half tables go to the point one above - e.g. for 4.5 tables, use 5)

    Use the Tables - button to get rid of empty tables.

    Click on the Start round 1 button.

    During the session¶

    You can see the progress of each table trick by trick in the table number icon in the table list

    The E indicates that the table has completed the board but is not ready to move on. If you think this is stuck you should visit the table and, if necessary force the board to finish.

    When more than half the tables have completed the round, the Move button is displayed and enabled for the Director. The name of the button shows which round it relates to.

    If some tables have not finished the round, their opponents in the following round will have to wait - just like live bridge.

    If a table seems to be struck, you can visit at director and if they have finished the round you can click on the End Round button to move them on.

    If a table is very slow you can visit the table and if you decide it is appropriate, you can skip a hand - they will receive an average score.

    It is nice to make an announcement prior to moving, allowing people to finish their conversations.

    You can visit any table at any time and not just when you are called.

    You can kick a player from a table to the lobby. Click on Advanced (in top bar), then under Remove Player select the player you want to eject and click on Kick player to lobby.

    Scoring¶

    Log in as director and click on the USBIO button at the top right. A new panel will appear

    With the mouse right-click on the USEBIO file and select Save link as …

    This should save a file with a name similar to 210325121204PhoenixBridgeClub.xml (Look for the xml extension - in Windows you might have to turn on extensions). In your Downloads directory. Check the date and time to make sure it is the file that you want.

    Similarly, right-click on *PBN file and save the file with a name similar to 210325121204PhoenixBridgeClub.pbn.

    Upload the files to BridgeWebs and the EBU following the instructions in Sending scores to Bridgewebs and Sending scores to EBU.

    Destroying the session¶

    Real Bridge ask us to destroy sessions as soon as reasonable after it has finished. DO NOT destroy the session until the scores have been extracted.

    Log in as director and click on the Advanced button at the top right. A new panel will appear

    To Destroy the session you must double-click on Destroy session. This is irreversable and so you must be satisfied that the scores** have been extracted.
