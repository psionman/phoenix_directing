    Uploading deal files¶

    This procedure describes how to upload deal files for a BBO tournament, for example, the Children in Need national duplicate event.

    1. Ensure that you have the file loaded onto your computer. It must be in the LIN format; not PBN.

    2. Log into BBO with the club’s director’s username (vebu2111328)

    3. Click on the Account tab on the right hand side

    4. Click on Deal archive in the top menu

    5. Click on the + against SELECT FOLDER. A new folder named Untitled folder will be created

    6. Click on the pencil icon next to this folder to edit the details

    7. Change Untitled folder to a meaningful name

    8. Click on the Import LIN file button at the bottom of the panel

    9. Click on choose file in the dialog box

    10. Click on the Import button. You will see a message confirming that the file has been uploaded

    11. Click on the Director tab on the right hand side

    12. Click on the Pending Tournaments tab and the tournament in question. Choose edit tournament from the drop down menu.

    13. In the Format tab, go to Deal source and click on Use random deals and select Use saved deals

    14. Click on the Select folder button and choose the folder that you created above

    15. Click on Select in the top right hand corner

    16. Click on modify tournament to save the changes

    The deal is now set for the tournament.
